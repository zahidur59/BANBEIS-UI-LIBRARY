BANBEIS UI GUIDE ICONS COMPONENTS

Font icons

Uses

Get all the BANBEIS icon fonts and "banbeis_fonticons.css" from https://github.com/zahidur59/BANBEIS-UI-LIBRARY.git then add the following line your page header - 

<link rel="stylesheet" href="css/banbeis_fonticons.css">

<span class="icon-Academic-cap"></span>
